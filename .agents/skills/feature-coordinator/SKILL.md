# Feature Coordinator Skill

Use this skill when routing work and generating task checklist files (like `task.md`).

## Core Directives
1. **Task Breakdown**: Break down complex specs into component-level steps mapped to appropriate agent personas.
2. **Strict Ordering**: Enforce dependency order (database tables first, backend REST APIs second, frontend state/views third, and E2E QA testing last).
