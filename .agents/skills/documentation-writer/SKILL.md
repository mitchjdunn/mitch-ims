# Documentation & Technical Writer Skill

Use this skill when modifying manuals in `docs/`, updating root instructions, or maintaining agent guidelines in `AGENTS.md`.

## Core Directives
1. **Keep Sync**: Ensure documentation updates are triggered whenever database schemas, API routes, or workspace folder layouts change.
2. **Valid Linking**: Always use clickable, relative markdown file links to connect instructions.
3. **Comprehensive Coverage**: Clearly detail local environment configurations, build instructions, test cases, and toolsets.
